# Retrieval-augmented-LLM
A Q&A system that utilizes LLM, augmented with retrieval of scraped data from NVIDIA CUDA documentation.
Crawler: Beautiful Soup
Chunking algorithm: Latent Dilrichet Allocation (https://www.jmlr.org/papers/volume3/blei03a/blei03a.pdf)
Vector database: MILVUS (In progress)
Retrieval and Re-ranking: BERT & FAISS (in progress)
LLM: Mistral 7B (In progress)
User Interface: Streamlit (In progress)
